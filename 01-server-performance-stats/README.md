# Server Performance Stats

A lightweight Bash-based server monitoring tool that displays real-time system performance metrics.  
This project is part of the **DevOps Roadmap practice** and helps build strong Linux administration and shell scripting skills.

## Features
- Shows CPU usage  
- Shows RAM usage  
- Shows Disk usage  
- Lists Top 5 CPU-consuming processes  
- Lists Top 5 RAM-consuming processes  
- Fully terminal-based and dependency-free  

## Usage

Make the script executable:

```bash
chmod +x server-stats.sh
```

Run the tool:

```bash
./server-stats.sh
```

## Purpose
This project helps beginners practice:
- Monitoring Linux systems  
- Working with core commands like `top`, `ps`, `df`, and `free`  
- Writing real-world Bash utilities
